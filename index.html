Frequently Asked Questions

Is MJ PDF Reader completely free? +
Yes! MJ PDF Reader is Free and Open Source Software (FOSS). There are no ads, no in-app purchases, and no hidden costs.
Does it support Dark Mode? +
Yes, the app features a powerful Dark Mode that not only changes the UI but can also invert the colors of the PDF document itself for comfortable night reading.
What is Text Mode? +
Text Mode extracts the text from a PDF file and displays it in a readable, reflowable format. You can adjust the font size and background color, making it perfect for reading on small screens.
Can I open online PDFs? +
Yes, MJ PDF allows you to open PDFs directly from web links. It requires the Internet permission solely for this function.
